Sauce for Zwift™ Event Control Hashtags
For Event Organizers Only: Use these tags to control Sauce behavior during your events.

Sauce will look for words resembling #hashtags in the event description of an event. When certain hashtags are present Sauce will change its behavior for any the athletes joined to that event.


Things to know:
  • They only affect athletes DURING the event
  • Some of the tags only apply to peer athletes (i.e. not the Sauce user themselves)
  • Using a different computer or fan-view will NOT circumvent this; a peer is determined by the Main Zwift login
  • You can put the hashtag anywhere in the description but generally it's best to put them towards the bottom
Currently supported tags:
  • #hidewbal: Hide W'bal information for peer athletes (your own is still available)
  • #hideftp: Hide FTP-related fields like TSS and Intensity Factor for peer athletes
  • #nooverlays: Disable Sauce for ALL athletes in this event, including themselves
  • #hidethehud: Same as #nooverlays but used internally by Zwift to control the game too
