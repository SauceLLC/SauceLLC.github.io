Other / More / I'm tired of screen capturing

Since I'd rather spend my time programming than maintaining goofy websites. Here are the other things worth mentioning..

  • Requirements
    1. Patreon membership
      • Let's be honest, this is pretty niche software. I'm just looking to make an honest wage and at present that means a very small monthly patronage is required.
      • However, here is your free-trial hack. Patreon doesn't bill until the 1st of the month. Subscribe, try Sauce, and cancel before the following 1st and you'll not be charged.
    2. A Zwift account - duh
    3. A second Zwift account - wait, what!?
      • Don't panic
      • This is technical detail and you do NOT need to pay for a second Zwift account.
      • Zwift's network doesn't allow multiple connections with the same login.
      • Sauce has its own connection, so it needs its own "monitor" account too. Because it only "Fan Views", you don't need active billing on this account.
      • You don't need to write all this down. Just Download Sauce and launch it. It will guide you through the process.
  • Mods
    • Short for modifications, they are security sandboxed packages modelled after browser extensions.
    • Code is only executed in a Web Browser context. As such, they only work with web tech like JavaScript, WASM, HTML and CSS.
    • Backend code execution is not allowed.
    • Browse existing Community Mods
    • Build your own from the GitHub Example Mod project
  • API / Web Access
    • Sauce runs an optional web server on your local network.
    • Default access is on port 1080, a la..
      • http://localhost:1080
      • https://localhost:1081 - TLS Encrypted (self signed cert)
    • The web server serves two functions:
      1. Local network access to Sauce Windows from phones, tablets, other same-network computers
      2. REST and WebSocket API for writing your own tools on top of Sauce
        • http://localhost:1080/api - REST starting point
        • Example S4Z WebSocket client
  • Data Usage
    • Sauce is a client-side application.
    • All Zwift data is exchanged directly between your computer and Zwift servers. No cloud storage, no proxying of data. It's just you and Zwift.
    • Sauce is built on Electron, which is built on Chromium, so in short, you're running a specialized web browser that understands the Zwift network protocols.
  • Restrict use
    • Event organizers wanting to disable or modify the use of Sauce for participants in their event, can add special hashtags to their event description.
    • To have your data filtered out send a SHA256 hash of your Zwift ID to support@sauce.llc and I'll add it to the block ledger.
